# 🌡️ Temperature Converter

A beginner-friendly temperature conversion web application built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Simple and responsive UI
- Instant conversion on button click
- Input validation for unit selection

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
